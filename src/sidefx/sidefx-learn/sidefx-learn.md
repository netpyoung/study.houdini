    - Attribute Randomize로 index를 설정하고
    - Copy - Stamp - Stamp Input 체크 및 Attribute Stamps에 index ?? 스탬프가 뭘까?

  - Assemble : 매쉬 정리

  - Blast로 원하는거 몇개 골라서 @P.z > 5

  - For-Each Connected Pieces
